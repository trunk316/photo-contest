# 写真投稿サイト（授業最終課題）

Webプログラミング授業の最終課題として制作した、PHPによる写真投稿サイトです。
ユーザー登録から写真の投稿、投票（いいね機能）までの一連の機能を備えています。

## 🛠️ 使用した技術
* HTML / CSS
* PHP (フレームワークなしのPHP)
* MySQL (データベース)
* Docker / Docker Compose (※ベースの環境構築用ファイルは学校配布のものを活用)

## ✨ 実装した機能
* **アカウント管理機能：** 新規登録、情報変更、退会（削除）
* **権限管理：** アカウントタイプ別（投稿者/閲覧者）の操作制限
* **写真投稿・閲覧機能：** 写真のアップロード、投稿一覧表示
* **インタラクション機能：** 写真への投票機能
* **データ管理：** ユーザー情報および写真データのMySQLによる管理

## 💡 設計・実装のこだわりポイント
* **効率的な画像管理：**
  画像データをそのままデータベース（MySQL）に保存すると容量が膨大になり、パフォーマンス低下を招くため、画像本体はサーバー内のフォルダに格納し、データベースにはそのファイルパスのみを記録する効率的な設計にしました。
